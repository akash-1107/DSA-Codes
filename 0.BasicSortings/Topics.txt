1.Sorting Techinques - 
            1.Bubble Sort ✅ TC = O(N^2) and SC = O(1)
            2.Selection Sort ✅ TC = O(N^2) and SC = O(1)
            3.Insertion Sort ✅ TC = O(N^2) and SC = O(1)
            4.Quick Sort  ✅ TC = O(N*LOGN) and SC = O(1) + O(N) recursion stack space
            5.Merge Sort  ✅ TC = O(N*LOGN) and SC = O(N) (extra array in merge part) + O(N) recursion stack space
